# Maincrafts Task 3 – Contact Form with JavaScript

## 📌 Project Overview
This project is developed as part of the Maincrafts Full Stack Web Development Internship – Task 3.

The website includes a Contact Form with JavaScript validation, LocalStorage support, and a Submissions page to display saved data.

## 🚀 Features
- Responsive Multi-Page Website
- Contact Form
- Client-side Form Validation
- Save Data using LocalStorage
- Display Submitted Data on Submissions Page

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage

## 📂 Pages
- Home
- About
- Contact
- Submissions
